This is a copy of my DnD ObsidianVault. This includes general DnD information as well as notes to the ongoing campaign "Buried beneath Graves" and obsidian specific folders such as  `./.obsidan/`. Feel free to click through it and let it inspire you.
# For people who know the campaign
For you, the most important folder is probably the campaign folder `./BuriedBeneathGraves/`. It includes all the information i was yet able to extract from the audio files as well as the cut audio itself. It does neither include the raw audio nor the audacity files due to file sizes.  
Everything is structured in its corresponding folder though it might be helpful to check [my Notes](./BuriedBeneathGraves/Notes.md) too, especially for newer information.
# Style
## Links
Two different types of links are used in this project. For existing files the normal markdown format of `[display name](path or url)` is used. For not yet existing files Obsidian specific links are used, indicate by double square brackets (`[[file name]]`). In such way it is easier to keep track of not yet created files while also keeping the the readability of the notes on an acceptable level.  
Folders are not directly linked but their relative path is written as inline code (`relative path`)
## Tags
Tags are a Obsidian feature similar to links to not created files. Tags are usually used for properties. This includes spell properties such as a spells school or range as well as the time of a day or other pre-definable values that do not need further explanation.  
For better readability, tags are only in the frontmatter of a file, not the body.  
Tags are always written in camelCase
All possible Tags are written down in [the index](./index.json).
## Language
This project is written in two languages, English and German. This is because the campaign is being played in German, most of the sources are written in English. The languages are strictly separated from each other except for two cases.  
**English:** English is used for the folder structure and Tag names as well as general DnD information such as Classes, Races, or Spells.  
**German:** German is used for the notes concerning the campaign itself, meaning notes for party members, locations, session summaries, etc..  
**Exceptions:** English and German can mix in [my notes](./BuriedBeneathGraves/Notes.md), because it is more a scrap paper than an actual information page, and in the character sheets (`./BuriedBeneathGraves/Party/CharacterSheets/)
## Folders and Files
Folders are strictly named using PascalCase.  
Files are named after what they represent and do not follow a strict multi-word identifier format. This can lead to whitespace in a file name.
